# Joseph's Anthropic Academy Plan

Strict 4-week (~30 hour) study plan for [Anthropic Academy](https://anthropic.skilljar.com/) — published as a GitHub Pages site.

## 🌐 Read it as a website

**[https://joseph19820124.github.io/anthropic-academy-plan/](https://joseph19820124.github.io/anthropic-academy-plan/)**

## What's in here

- **[index.md](index.md)** — overview, course tiering, hard rules
- **[PLAN.md](PLAN.md)** — day-by-day schedule with checkboxes
- `_config.yml` — Jekyll config (Cayman theme)
- `artifacts/` — per-week deliverables go here as I complete them

## How I use this repo

```bash
# Every day:
#   1. Open PLAN.md, find today's row
#   2. Do the lesson + exercise
#   3. Tick the checkbox in PLAN.md
git -c user.email="joseph.siyi@gmail.com" commit -am "day N done: <course>"
git push
```

The commit graph is the progress dashboard. No commits for 2+ days = re-baseline.
